# Practica 1 - creación de clases y encapsulamiento en php

## Objetivo de la practica
aplicar los conocimientos aprendidos de la programacion orienteda a objetos, para crear una clase usuario con encapsulamiento.

## Descripción 
se creó una clase llamada Usuario que contiene atributos privados como lo son nombre y correo, tambien se incluyeron metodos como getters y setters para los valores.

## Ejecución
1. se coloco la carpeta practica 1 dentro de la carpeta htdocs del servidor local xammp.
2. corremos apache y accedemos al localhost en el navegador (https://localhost/desarrollo-web-avanzado-fimaz-uas/parcial-1-poo/practica-1/index.php)
3. visualizamos los resultados
