# Práctica 4 - Integración POO + Herencia + Excepciones

## Objetivo
Desarrollar un sistema simple en PHP usando Programación Orientada a Objetos, aplicando herencia, validación de datos y manejo de excepciones.

## Requisitos
- PHP 8+
- XAMPP
- Navegador web

## Ejecución

Abrir en el navegador:

http://localhost/desarrollo-web-avanzado-fimaz-uas/parcial-1-poo/practica-4/index.php

## Evidencia esperada

El sistema muestra:

1. Una tabla HTML con usuarios:
- Administrador
- Alumno
- Invitado

2. Un error controlado cuando se intenta crear un usuario con correo inválido.
