# Practica 1 - creación de clases y encapsulamiento en php

## Objetivo de la practica
El propósito de este ejercicio es que el estudiante comprenda no solo cómo funciona un CRUD con PHP-
PDO, sino también cómo estructurarlo correctamente desde una perspectiva profesional, aplicando buenas prácticas de desarrollo web y diseño de software.
## Descripción 
En este sistema, cada producto se maneja como un objeto. La conexión a la base de datos se encapsula en una clase independiente para reutilizarla. El CRUD se concentra en un controlador que usa PDO con consultas preparadas para trabajar de forma segura. Finalmente, la página principal permite capturar y mostrar información, integrando el formulario y la tabla en una sola interfaz.

## Ejecución
1. se coloco la carpeta examen-practico dentro de la carpeta htdocs del servidor local xammp.
2. creamos la base de datos en el servidor locar
3. corremos apache y accedemos al localhost en el navegador (https://localhost/desarrollo-web-avanzado-fimaz-uas/parcial-2-pdo/)
4. visualizamos los resultados
